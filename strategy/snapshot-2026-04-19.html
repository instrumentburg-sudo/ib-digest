Горизонт 2028 — единственный вопрос без ответа

За последние недели вся работа и весь knowledge — тактика: SLA, автопилот заказов, event-sourced измерения, процессы приёмки. Ни одного brainstorm «каким ИнструментБург будет через 3 года». Цель «1 млн чистыми» без выбранного сценария — якорь, а не задача.

01 · Кормушка

Выручка ~1,5 млн стабильно, автоматизация до потери пульса, Антон 20 ч/нед, прибыль 300–400k.

02 · B2B-платформа

Убираем физиков, 10–15 корпоратов по подписке. «Ремонт-как-услуга» для промышленности.

03 · Актив на продажу

18–24 мес упаковки — выход с капиталом 20–40 млн. Всё, что делаем сейчас, становится инвестиционным аргументом.

04 · Плацдарм

Сервис кормит, пока строится что-то новое: SaaS для СЦ, новый продукт, смежная ниша.

Ни один из четырёх ещё не выбран. Пока не выбран — любой тактический совет укрепляет одну из моделей, которая может оказаться не той.
