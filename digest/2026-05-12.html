📡 Что нового в AI:

UI-TARS-desktop выстрелил в трендах GitHub

Что случилось multimodal desktop agent stack от ByteDance стал одним из самых заметных AI-репозиториев дня. Это уже не игрушка, а готовая база для desktop-агентов с упором на реальное управление интерфейсом.

Живой пример https://github.com/bytedance/UI-TARS-desktop

👉 Как применить: смотреть как референс для наших сценариев браузерной и десктопной автоматизации, особенно для сложных SPA и операторских интерфейсов.

MartinLoop — «поводок» для кодящих агентов

Что случилось на r/ClaudeAI обсуждают open source control plane для AI coding agents с жёсткими лимитами бюджета, kill-switch, audit log и verifier-этапами. Это реакция на реальную боль: агенты жгут токены и уходят в бесконечные циклы.

Живой пример https://github.com/Keesan12/Martin-Loop

👉 Как применить: взять идеи лимитов по деньгам/итерациям и обязательной верификации перед финалом для наших длинных агентных задач и cron-джобов.

Sandra даёт Claude долговременную память через MCP

Что случилось на r/ClaudeAI показали graph+vector memory backend, где факты пишутся как связи и потом нормально извлекаются между сессиями. Есть MCP-подключение и демо.

Живой пример https://github.com/everdreamsoft/sandra

👉 Как применить: подсмотреть модель графовой памяти и способы хранения фактов/связей для развития нашей memory-архитектуры без разрастания плоских заметок.

OpenGravity — браузерная среда для vibe coding без установки

Что случилось на Hacker News обсуждают open source клон Antigravity: чистый браузер, свой ключ, терминал и быстрый запуск без локальной настройки.

Живой пример https://opengravity.pages.dev

👉 Как применить: сильная идея для лёгких одноразовых рабочих сред — быстро поднимать песочницу под задачу, демо или проверку гипотезы без полноценного деплоя.

AI за вечер собрал систему, которая нашла причину ночных пробуждений

Что случилось на Hacker News обсуждают реальный кейс, где coding agent за несколько часов собрал связку Raspberry Pi + микрофоны + Home Assistant + дашборд, чтобы понять, что будит человека ночью.

Живой пример https://martin.sh/i-let-ai-build-a-tool-to-help-me-figure-out-what-was-waking-me-up-at-night

👉 Как применить: это хороший шаблон для наших прикладных микро-систем — агент быстро собирает узкий инструмент вокруг конкретной боли вместо долгой ручной разработки.
