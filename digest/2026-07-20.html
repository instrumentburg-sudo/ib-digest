Claude Code уже месяц работает на Bun, переписанном на Rust

Что случилось 19 июля Саймон Уиллисон проверил бинарник Claude Code и подтвердил, что начиная с версии 2.1.181 внутри используется Rust-версия Bun. По данным автора Bun, запуск на Linux ускорился примерно на 10%. На момент сбора обсуждение на Hacker News набрало 463 балла и 619 комментариев.

Живой пример https://simonwillison.net/2026/Jul/19/claude-code-in-bun-in-rust/

👉 Как применить: отдельной миграции не нужно. Для серверных заданий Claude Code можно просто обновлять штатно; выигрыш будет заметен прежде всего на частых холодных запусках.

Code Review Graph резко вырос в GitHub Trending

Что случилось репозиторий получил 663 звезды за сутки и дошёл до 21 709. Инструмент строит локальный граф кода через Tree-sitter и SQLite, а затем отдаёт агенту через MCP только затронутые файлы и зависимости. Заявленные авторами тесты показывают сокращение контекста в 38–528 раз, но это ещё надо проверять на наших репозиториях.

Живой пример https://github.com/tirth8205/code-review-graph

👉 Как применить: провести пробный разбор одного изменения в репозитории InstrumentBurg — обычным способом и через граф. Сравнить расход контекста, время и пропущенные зависимости; только после этого подключать постоянно.

Wigolo — локальный поиск и сбор данных для агентов без платного API

Что случилось проект получил 595 звёзд за сутки и вырос до 2 025. Он объединяет поиск, загрузку страниц, обход сайтов, извлечение данных и кэш через MCP; базовые функции работают локально без ключей. Проект пока в открытой бета-версии, требует Node.js 20+ и около 1,5 ГБ на диске.

Живой пример https://github.com/KnockOutEZ/wigolo

👉 Как применить: проверить как резервный источник для ежедневного дайджеста — сегодня облачные web_search и web_extract не работали без Firecrawl. Сначала прогнать на отдельной машине и сравнить полноту, свежесть и количество блокировок.

Headroom сжимает контекст перед отправкой агенту

Что случилось в X разошёлся открытый слой для сжатия журналов, ответов инструментов, JSON и фрагментов базы знаний. Авторы заявляют экономию 60–95% токенов на JSON и 15–20% в задачах программирования; есть локальный прокси и MCP, исходные данные можно вернуть из кэша.

Живой пример https://github.com/headroomlabs-ai/headroom

Источник https://x.com/xnone_l/status/2079084210942341238

👉 Как применить: испытать на одном длинном задании со сводкой или журналами. Критерий — реальная экономия токенов без потери чисел и ссылок; оборачивать все рабочие агенты до такого теста рано.
