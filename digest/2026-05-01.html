🔧 OpenClaw v2026.4.29:

  • Вышел свежий релиз OpenClaw: упор на управление активными запусками, видимые ответы в мессенджерах, метаданные маршрутизации подагентов и opt-in follow-up commitments для напоминаний через heartbeat.

Jcode — лёгкий harness для параллельных coding agents

Что случилось в GitHub Trending поднялся 1jehuang/jcode — Rust-инструмент для multi-session coding workflows. Автор делает ставку на быстрый старт, низкое потребление памяти, граф памяти, side panel, агентные swarm-сценарии и уведомления между агентами, если код меняется «под ногами».

Живой пример https://github.com/1jehuang/jcode

👉 Как применить: подсмотреть идеи для наших исполнителей: лёгкие сессии, явные каналы между агентами, авто-уведомления о конфликтующих правках и граф памяти без постоянного расхода контекста.

Что случилось obra/superpowers продвигает полный процесс для coding agents: сначала уточнение цели и дизайн, потом worktree, план, TDD, подагенты, ревью и только потом завершение ветки.

Supply-chain атака на PyPI lightning бьёт по AI-разработчикам

Что случилось Semgrep сообщил о компрометации PyPI-пакета lightning версий 2.6.2 и 2.6.3 от 30 апреля. Вредоносный код крадёт токены/секреты, пытается распространяться через npm и добавляет persistence через .claude/settings.json и .vscode/tasks.json.

Живой пример https://semgrep.dev/blog/2026/malicious-dependency-in-pytorch-lightning-used-for-ai-training/

👉 Как применить: проверить проекты с ML/LLM-зависимостями на lightning==2.6.2/2.6.3, неожиданные .claude/setup.mjs, .claude/router_runtime.js, .vscode/setup.mjs, .vscode/tasks.json и подозрительные workflow. Если пакет ставился в CI — считать окружение скомпрометированным и ротировать токены.
